В процессе работы над проектом "Тестирование интернет-магазина" предоставляется важный опыт практической отработки навыков ручного тестирования. В рамках данного проекта предстоит работа "над ошибками" в разработке сайта, поиск и исправление багов, составление тест-кейсов, регрессионное тестирование, описание процесса тестирования. Так же в репозитории можно найти информацию с файлом "обо мне", непосредственно данный файл с кратким описанием проекта. Демонстрационная версия сайта интернет-магазина, располагается по адресу: https://products-store.hexlet.app/. Данный сайт представляет собой каталог товаров с возможностью сортировки по категориям, стоимости, производителю товара, наличию в продаже. У тестируемого магазина есть возможность поиска товара по названию, есть несколько вариантов языковой трансляции, продукцию можно добавлять в корзину, удалять из корзины, оформлять заказ. Проект создается на базе навыков, полученных при обучении в онлайн-школе Hexlet. Наставник, консультирующий по вопросам обучения: Фадеева Александра. Автор проекта: Асонова Анна.
